# C语言学习路线

## 怎样高效地学习C语言？

1. 多阅读优秀项目的源码
2. 多借鉴优秀项目的思想
3. 多修改自己完成的代码
4. 编写的代码要多上机测试
5. 养成良好的编程风格

## C语言要学习什么?
### 指针
- 函数指针
- 回调函数
- 数组与指针
- 多级指针
- 指针声明

书籍推荐
- 《C和指针》
- 《深入理解指针》

### 内存
- 堆
- 栈
- 虚拟地址
- 内存对齐
- Linux C程序内存布局
- 动态内存
  - 内存池
  - malloc, tcmalloc, jemalloc, ptmalloc
  - free

书籍推荐
- 《程序员的自我修养》
- 《CSAPP》
  - 第3章节
  - 第7章节
  - 第10章节

### 编译和链接
- 静态链接
- 动态链接
- 符号解析

书籍推荐
- 《程序员的自我修养》
- 《CSAPP》
  - 第7章节
- 《loader and linker》

### 函数
- 函数调用栈
- 参数传递
- 返回值

### 汇编
《汇编语言》王爽
